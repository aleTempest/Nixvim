# Run this config

1. Inside the repository:
``` bash
nix run .
```

2. Or you can run it remotely:
```bash
nix run 'github:aleTempest/Nixvim'
```

# Supported Languages
- Lua
- Python
- Nix
- Latex
    - You can change the spell lang in `./config/languages/latex`. Default is eng.

LSP, autocompletion and syntax highlighting.
